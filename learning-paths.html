How Learning Paths Work

Four steps from picking a path to landing an interview call.

01
Pick a Path
Choose a path matching your current level. Each card shows difficulty, estimated weeks, and the technologies you'll learn.
02
Work the Checklist
Open the path and tick off topics one by one. Every topic has curated links — video, article, or official docs. Your progress saves automatically.
03
Build the Project
Every path ends with a real capstone project. Push it to GitHub, write a README, and add it to your portfolio for recruiters to see.
04
Stack & Repeat
Paths are designed to build on each other. HTML → JS → Bootstrap → full project. Each one completed makes the next one faster.
